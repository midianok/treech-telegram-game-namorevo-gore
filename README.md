# Treech Flappy

Простая HTML5-игра в стиле Flappy Bird на Phaser, TypeScript и Vite.

## Команды

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Управление

- `Space` или `Arrow Up` - взмах
- клик или тап - взмах
- после проигрыша `R`, `Space`, клик или тап - рестарт

Лучший счет сохраняется в `localStorage`.
